# Digital Warranty & Bill Management System

This project helps users store bills digitally and track warranty expiry dates.

## Features
- User Registration & Login
- Add Bills with Images
- Track Warranty Expiry
- Download Bills
- Clean UI Dashboard

## Technologies Used
- Python (Django)
- HTML, CSS
- SQLite
